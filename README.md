# Jenkins Groovy Pipeline Examples

A beginner friendly repository to learn Jenkins Pipelines using Groovy DSL.

This repository contains simple Jenkinsfile examples that gradually introduce Jenkins Pipeline concepts.

## Learning Objectives

By completing these examples, you will learn:

- Declarative Pipeline syntax
- Stages and steps
- Environment variables
- Build-Test-Deploy workflows
- Conditional execution
- Groovy functions in Jenkins
- Code reuse techniques
