# Jenkins Groovy Pipeline Examples

A beginner friendly repository to learn Jenkins Pipelines using Groovy DSL.

This repository contains simple Jenkinsfile examples that gradually introduce Jenkins Pipeline concepts.

## Learning Objectives

By completing these examples, you will learn:

- Declarative Pipeline syntax
- Stages and steps
- Environment variables
- Build-Test-Deploy workflows
- Conditional execution
- Groovy functions in Jenkins
- Code reuse techniques

## Examples

| Example | Topic                     |
| ------- | ------------------------- |
| 01      | Hello World Pipeline      |
| 02      | Environment Variables     |
| 03      | Build Test Deploy         |
| 04      | Conditional Pipeline      |
| 05      | Functions and Reusability |

## Requirements

- Jenkins installed
- Git
- Basic command line knowledge

## How to Use

1. Clone this repository

```bash
git clone https://github.com/km-saifullah/jenkins-groovy-pipeline-examples.git
```

2. Create a Jenkins Pipeline Job
3. Point Jenkins to this repository
4. Select the Jenkinsfile you want to execute
5. Build the pipeline and inspect the logs
